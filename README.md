# Polymer workshop
# DEMO - Talking with the map, listening to your weather 

This demo app:
* consumes [Open Weather Map API] (http://openweathermap.org/api) to obtain data about weather in a city
* speak (your weather)
* listen (your city)


## Installation
~~~
npm install
bower install
~~~

## Execution
~~~
grunt serve
~~~
